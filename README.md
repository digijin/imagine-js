# ImagineJS

An component based framework for highly browser compatible interactions

To compile (currently needed for anything to work):

 * install nodejs
 * run:
```
npm install
webpack
karma start
 ```

### Features

 * Input manager
 * Time manager
 * premade components
  * box collision detection
  * absolute element positioning
 * demos
 * docs
